ACERCA DE EFLOOD²

ANÁLISIS
HIDRÁULICO

eFlood² es una aplicación de escritorio moderna para el análisis avanzado de modelos hidráulicos HEC-RAS 2D. Desarrollada con Tauri, React y Python, ofrece una interfaz intuitiva para procesar archivos HDF5, visualizar datos hidráulicos y exportar resultados a VTK.

CARACTERÍSTICAS

CAPACIDADES
AVANZADAS

Análisis HDF5

Procesamiento completo de archivos HEC-RAS con pyHMT2D. Análisis de hidrogramas, condiciones de contorno y valores de Manning.

Visualización 3D

Renderizado avanzado de simulaciones hidráulicas con Three.js. Visualización interactiva de flujos y superficies de agua.

Exportación VTK

Compatible con ParaView y otros visualizadores científicos. Exportación de mallas, vectores de velocidad y campos escalares.

Alto Rendimiento

Arquitectura Rust + Python optimizada para procesamiento rápido. Interfaz moderna con animaciones fluidas y respuesta inmediata.

Multiplataforma

Disponible para Windows, macOS y Linux. Instalación simple con dependencias automáticas.

RAS Commander

Integración con automatización de HEC-RAS. Análisis de flujo y condiciones hidráulicas avanzadas.
